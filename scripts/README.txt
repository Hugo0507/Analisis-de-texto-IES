SCRIPTS DE INSTALACION AUTOMATICA - PYTHON 3.11
================================================================================

ESTOS SCRIPTS TE AYUDARAN A MIGRAR DE PYTHON 3.13 A PYTHON 3.11

================================================================================
INSTRUCCIONES RAPIDAS:
================================================================================

1. EMPIEZA AQUI:
   - Ejecuta: EJECUTAR_AQUI.bat
   - Sigue las instrucciones en pantalla

2. DESPUES DE INSTALAR PYTHON 3.11:
   - Ejecuta: CONTINUAR_INSTALACION.bat

================================================================================
SCRIPTS DISPONIBLES:
================================================================================

EJECUTAR_AQUI.bat
  └─> Script maestro que inicia todo el proceso

CONTINUAR_INSTALACION.bat
  └─> Continua la instalacion despues de instalar Python 3.11

1_verificar_python.bat
  └─> Verifica tu sistema actual

2_limpiar_entorno.bat
  └─> Limpia el entorno virtual viejo

3_instalar_todo.bat
  └─> Instala todas las dependencias

4_configurar_env.bat
  └─> Configura variables de entorno

5_probar_app.bat
  └─> Prueba que la aplicacion funcione

instalar_dependencias.bat
  └─> Instalacion alternativa paso a paso
